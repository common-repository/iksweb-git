== Description ==

Плагин добавляет кнопку-информер в админ. панель в публичной части, с помощью которой можно узнать текущую ветку git, а так же изменить ее. Это функция очень поможет менеджерам и владельцам сайта, которые не работают с Git напрямую. 
В админ. панели вы сможете отслеживать логи/ветки и проверять есть ли незакомиченные правки. 
Инструкция по установки Git поможет быстро установить и коректно настроить гит для коректной работы на сайте.

== Frequently Asked Questions ==

= Что такое GIT? =

Git — распределённая система управления версиями. Проект был создан Линусом Торвальдсом для управления разработкой ядра Linux, первая версия выпущена 7 апреля 2005 года.

= Гости могут переключить ветку? =

Нет. Переключение доступно только для администраторов.

== Installation ==

Для установки необходимо наличие папки .git в корне сайта, а также необходима php функция exec, она должна быть включена.
После установки, в публичной части сайта, в админ. панели появится новая кнопка с информацией о текущей ветке git. Нажав на кнопку можно изменить ветку.
Для переключения ветки, git должен возвращать "nothing to commit, working directory clean".

1. Установить и активировать плагин;
2. В настройках плагина включите активацию;
3. После сохранения плагин начнёт автоматически отображать ветки GIT, если они есть;
